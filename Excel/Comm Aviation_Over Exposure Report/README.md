# Comm Aviation – Over Exposure Report (Américas)

This project contains the **credit risk exposure analysis** for the Commercial Aviation portfolio as of 30th September 2025.

## Project Files

- **Excel File:** [Download Comm Aviation Excel](./Comm%20Aviation_Over%20Exposure%20Report/Comm%20Aviation_Over%20Exposure%2030th%20September%2025%20-%20Américas.xlsx)
- **Documentation:** README.md (this file)

## Project Description

This Excel workbook:

- Aggregates and validates commercial aviation exposure data.
- Calculates over-exposure metrics per region and portfolio segment.
- Generates structured tables and dashboards for internal reporting.
- Provides insights for decision-making regarding credit limits and risk management.

## How to Use

1. Open the Excel file in Microsoft Excel (version 2016 or later recommended).  
2. Navigate through the different sheets to see data, metrics, and dashboards.  
3. Follow any notes or instructions included in the workbook for updates or recalculations.  
4. No external scripts or software are required; all calculations are performed within Excel.
